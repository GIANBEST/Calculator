# Calculadora

Bem-vindo ao projeto **Calculadora**! Este é um aplicativo simples e funcional para realizar operações matemáticas básicas.

## 📋 Funcionalidades

- Adição.
- Subtração.
- Multiplicação.
- Divisão.
-Raiz Quadrada.
-Modo Cientifico.

## 🚀 Tecnologias Utilizadas

JAVA e Gradle.
